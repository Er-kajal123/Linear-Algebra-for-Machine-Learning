# Linear-Algebra-for-Machine-Learning

This repository contains my hands-on practice of Linear Algebra concepts using Python and NumPy while preparing for Machine Learning.

## Topics Covered

### Session 1-3

* Vectors
* Matrices
* Images as Matrices
* Text Encoding
* Audio and Video Representation
* Dot Product
* Cosine Similarity
* L1 and L2 Norms

### Session 4-6

* Linear Dependence and Independence
* Vector Spaces
* Subspaces
* Span
* Basis
* Standard Basis
* Matrix Types
* Matrix Operations
* Matrix Multiplication

### Session 7-8

* Matrix-Vector Multiplication
* Matrix-Matrix Multiplication
* Row View and Column View
* Systems of Linear Equations
* Homogeneous Systems
* Non-Homogeneous Systems
* Rank
* Column Space
* Ax = b
* Xw = y in Machine Learning

## Key Learning

One of the most important realizations from this journey is that Machine Learning is fundamentally built on Linear Algebra.

Concepts such as matrix multiplication, linear combinations, rank, column space, and systems of equations directly appear in:

* Linear Regression
* Neural Networks
* PCA
* Recommendation Systems
* Optimization Algorithms
